PSU Tuition Calculator

Calculates tuition and fees as provided by Portland State University

Download this project as a .zip file Download this project as a tar.gz file

The purpose of this project is to use the data available in the pdf file "BAO_PSU_2011_2012_FeeBook.pdf" and create a tuition calculator for all majors and levels included in the file. The original file can be obtained from PSU's website at this link.

The calculator was created using VB.NET and Visual Studio 2010. Most of the code is located at ~/Code/WP1/WP1/Form1.vb.
